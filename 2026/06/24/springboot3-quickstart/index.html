一、环境准备

  • JDK 17+：Spring Boot 3 要求 Java 17 及以上版本
  • Maven 3.6+ 或 Gradle 7.5+
  • IDEA（推荐 IntelliJ IDEA 2022+）

  • SpringApplication.run()：启动内嵌 Tomcat，初始化 Spring 容器

七、常见问题

Q: 启动报 “java: 错误: 无效的源发行版: 17”
→ IDEA 中 File → Settings → Build → Compiler → Java Compiler，将模块 target bytecode 设为 17。

Q: 端口被占用
→ 修改 server.port 或使用 --server.port=9090 启动。
